Malware Detection Using Hybrid Method (Static and Dynamic Features)

Supervisor: Annajiat Alim Rasel | Sabbir Ahmed | Brac University

MY undergraduate thesis presents a robust, interpretable Android malware detection system. It leverages a hybrid approach combining static and dynamic feature analysis with advanced ensemble machine learning, including Random Forest, XGBoost, and CatBoost, significantly outperforming traditional Logistic Regression. The Random Forest model demonstrated superior performance, achieving over 98% accuracy across key metrics for static features, and over 96% for dynamic features, validating the efficacy of integrating both analysis types. Furthermore, the application of SHAP values provides critical interpretability, offering insights into feature contributions and enhancing trust in the model's decision-making process for real-world cybersecurity applications. This work advances the field by offering a highly accurate, scalable, and explainable solution to evolving Android malware threats.
